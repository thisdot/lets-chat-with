# Tech Stack

This application is built using the following core technologies:

- Nx
- Angular 14
- NgRx 14 (Store + Effects + DevTools)
- Amplify (Auth + Storage + Hosting)
- Amplify Angular
- Angular Testing Library (Karma + Jasmine)
- Serverless
- Cypress.io
- Prettier
- Storybook (w/Figma Addon)
- Transloco (i18n)
